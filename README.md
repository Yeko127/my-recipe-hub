Project Overview
Recipe hub is a responsive recipe search application built with react, vite and tailwindcss,powered by the Spoonacular API.
Users can search for recipes, view detailed instructions and add their own recipes locally and navigate between pages using client-side routing.

This project demonstrates modern frontend development practices including API intergration, responsive design, componnet-based architecture, and secure environment variable management.

OBJECTIVES
This project demonstrates understanding of:
1. React functional compononents
2. React hooks (useState, useEffect, useParams)
3. Client-side routing with react router.
4. API consumption using axios.
5. Environmental variable management in Vite
6. Responsive UI design using tailwind CSS
7. Production development with netlify

TECHNOLOGY USED
React
Vite
Tailwind CSS
React Router DOM
Axios
Spoonacular API
Netlify

CHALLENGES FACED
Managing clinet-side routing without nested <Router> errors
Securing APIs keys using vite environmental variables.
Handling asynchronous API data fetching and loading states.
Ensuring proper deployment configuration for SPA routing on Netlify.
